# Project Scope v1

## Goal

Define the initial scope for an RL + LLM diagnostic framework that can:

- run training jobs
- summarize training outcomes
- call an LLM for diagnosis and suggestions
- enforce safety and policy constraints
- retain useful history across iterations
- control the outer optimization loop

## Initial Deliverables

- training execution entrypoint
- metrics summarization step
- LLM diagnosis interface
- constraint validation layer
- history memory management
- loop orchestration
